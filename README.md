## Git vs GitHub

### Create repository

```
git init
```

### Create file README.md

```
git add README.md
```

### Create commit

```
git commit -m "first commit"
```

### Create branch

```
git branch -M main
```

### Managing remote repositories 

```
git remote add origin https://github.com/username/repository_name.git
```

### Pushing to branch

```
git push -u origin main
```
